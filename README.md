The best note taker and task manager.

## Install

    $ git clone https://github.com/cblp/ff.git
    $ stack install

## Usage

### Get some unfinished tasks

    $ ff
    1: buy milk

Or, if no tasks

    $ ff
    nothing

### Add new task or note

    $ ff new 'learn Haskell'

    $ ff
    1: buy milk
    2: learn Haskell
